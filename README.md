# CPU-Design-VHDL

A simple CPU design implemented in VHDL as a final project for Digital systems analysis course.

## Components
- **ALU** – Arithmetic Logic Unit
- **Control Unit (CU)** – Manages instruction execution
- **RAM** – Memory unit
- **Program Counter (PC)** – Tracks current instruction
- **Accumulator** – Stores computation results

## Simulation
Simulated using ModelSim.
